  <files>core/forge/renderers/topic_clip.py</files>
  <action>
Create the new module `core/forge/renderers/topic_clip.py`. Implement three functions:

1. `_detect_has_video(src: str | Path) -> bool` — run `ffprobe -v error -select_streams v:0 -show_entries stream=codec_type -of csv=p=0 {src}`; return True if stdout is non-empty. Mirror the exact subprocess pattern used by `multiply._has_audio()` (read it first — same ffprobe invocation shape, capture_output, text=True). Audio-only `.mp3` returns False; `.mp4` returns True.

2. `_cut_segment(src, start_s, end_s, out_path, has_video: bool) -> Path` — cut ONE segment, ALWAYS re-encoding (never `-c copy` — copy uses keyframe seek and bleeds 1–2s of the previous segment into speech). Seek before -i for fast seek, then -t for duration:
   - Video path: `ffmpeg -y -v error -ss {start_s} -i {src} -t {end_s-start_s} -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920,fps=30,format=yuv420p" -c:v libx264 -preset veryfast -crf 23 -c:a aac -b:a 192k -ar 44100 -ac 2 {out_path}` (out `.mp4`). Reuse the EXACT scale/crop filter string from `film_montage._cut_segment` (line ~71) so 9:16 framing matches the other formats.
   - Audio-only path: `ffmpeg -y -v error -ss {start_s} -i {src} -t {end_s-start_s} -vn -c:a aac -b:a 192k -ar 44100 -ac 2 {out_path}` (out `.m4a`). No video filter.
   Raise RuntimeError with the last 500 chars of stderr on non-zero exit (same error pattern as film_montage concat at line ~173).

3. `enforce_duration(segments: list[dict], min_s: float = 10.0, max_s: float = 60.0) -> list[dict]` — sum `(s['end_s'] - s['start_s'])` across all segments. If total < min_s, raise ValueError(f"total segment duration {total:.1f}s < {min_s}s minimum"). If total > max_s, return a copy with the LAST segment's end_s trimmed down by the overage (floor at start_s + 1.0s). Never mutate the input list. This MUST run before any cutting (pitfall 6: guard before concat, not after).

Module header: `from pathlib import Path`, `import subprocess`. Keep functions pure/stateless — no global I/O at import.
